One-Time Setup (on both Laptop & PC)
1. Configure Git Identity
git config --global user.name "Codelise"
git config --global user.email "your_email@example.com"

2. Make Sure Remote URL is Correct
Inside your repo folder:
git remote -v
If it shows something wrong (like Aeryun), reset it:
git remote set-url origin https://github.com/Codelise/remastering-js-fundamentals.git

3. Check/Reset Credentials (Windows)
If wrong credentials are cached:
git credential-manager erase
Enter:
protocol=https
host=github.com
(then hit Enter twice)
Next push will ask you to log in with Codelise.
